This program is a small employee database that can be manipulated through various commands including:
1) Add
2) Search
3) Remove
4) Display

Included are 3 files:
- database.txt
    - This file contains all of the employees and their details
- functions.py
    - This file contains all of the functions written for the program
- main.py
    - This file handles the functions and allows the user interaction with the program

To start type:

python3 main.py
