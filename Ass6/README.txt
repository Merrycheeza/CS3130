This program grabs stock information off of http://www.theglobeandmail.com/globe-investor/ and displays it for the user.
The user is given a choice of different stocks including:
1) S&P TSX
2) S&P 500
3) Dow Jones
4) Nasdaq
5) CAD/USD
6) Gold
7) WTI Crude

Included are 2 files:
- functions.py
    - This file contains all of the functions written for the program
- main.py
    - This file handles the functions and allows the user interaction with the program

To start type:

python3 main.py
